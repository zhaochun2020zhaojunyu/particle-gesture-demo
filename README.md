# 粒子手势交互演示

一个基于 Three.js + MediaPipe Hands 的交互式粒子系统，支持手势控制粒子爆炸与还原。

## 功能特性

- 🪐 **三种粒子形态**: 土星、爱心、五芒星
- 🖐️ **手势识别**: 张开手掌 = 爆炸，握拳 = 还原
- 🎮 **演示模式**: 无需摄像头，自动循环展示
- ⚙️ **实时调节**: 爆炸强度、粒子大小、过渡速度
- 📱 **响应式设计**: 支持桌面和移动端

## 技术参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 粒子数量 | 15,000 | 平衡性能与视觉效果 |
| 目标帧率 | 60fps | 中端显卡可流畅运行 |
| 手势延迟 | ~300ms | 10帧确认 + EMA平滑 |

## 运行方式

### 方法1: Python HTTP 服务器
```bash
cd particle-gesture-demo
python -m http.server 8000
```
然后访问 http://localhost:8000

### 方法2: Node.js serve
```bash
cd particle-gesture-demo
npx serve
```

### 方法3: VS Code Live Server
安装 Live Server 插件，右键 index.html 选择 "Open with Live Server"

## 使用说明

1. **允许摄像头权限** - 用于手势识别
2. **手势控制**:
   - 🖐️ 张开手掌 → 粒子爆炸散开
   - ✊ 握紧拳头 → 粒子还原成形态
3. **点击底部按钮** - 切换土星/爱心/五芒星形态
4. **右侧参数面板** - 调节爆炸强度、粒子大小、过渡速度
5. **演示模式开关** - 无摄像头时自动循环展示

## 算法说明

### 形态采样
- **土星**: 斐波那契球面采样 + 圆环采样
- **爱心**: 参数方程 `x=16sin³t, y=13cost-5cos2t-2cos3t-cos4t`
- **五芒星**: 五角星轮廓 + 径向填充

### 手势判定
- 指标: 指尖到手腕平均距离 / 手掌大小
- 阈值: Open > 0.65, Close < 0.45
- 平滑: EMA (α=0.3) + 10帧确认

### 粒子动画
- 爆炸: 向外速度 + 涡旋噪声
- 还原: 弹簧阻尼向目标点移动
- 过渡: 动态插值，无跳变
